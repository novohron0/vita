Включи автообновление

Открой Команды → вкладка «Автоматизация» → + → «Время суток» → поставь 00:05, каждый день → «Выполнять сразу» → выбери ярлык Vita.

Это единственный шаг руками — Apple не разрешает включить расписание за тебя. Зато потом всё работает само, навсегда.

Поставить обои сейчас

Кнопка запустит ярлык — обои встанут сразу, не дожидаясь ночи.

Не дружишь с Ярлыками? Просто открой картинку, сохрани в Фото и поставь как обои. Но обновлять придётся руками — с ярлыком точки закрашиваются сами.

Эффект «часы прячутся за объект» тут не появится — iOS включает его только для фото с распознанным объектом (горы, человек) и только при ручной установке из Фото. К обоям, которые ставит автоматика, Apple его не применяет.
