## TestFlight / App Store (черновик)

1. Apple Developer $99/год
2. Xcode → Product → Archive → Distribute → App Store Connect
3. App Store Connect → TestFlight → добавить тестеров
4. На iPhone: TestFlight → установить Vita Focus
5. Настройки → Safari → Расширения → Vita Focus

Скриншоты для Store: popup, YouTube до/после, лендинг vitadots.ru/focus.
