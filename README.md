# Vita — твоя жизнь в точках

Генератор обоев для айфона: месяц / год / жизнь в виде сетки точек.
Каждый день сервер закрашивает новую точку — обои обновляются сами через Ярлыки iOS.

## Запуск

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
.venv/bin/uvicorn app.main:app --port 8787
```

Открыть http://localhost:8787

## Как устроено

- `static/` — одна страница: настройка + живое превью на канвасе + скачивание PNG
- `app/render.py` — тот же рендер на Pillow для серверных картинок
- `POST /api/link` — сохраняет настройки, возвращает персональную ссылку + ссылку на установку
- `GET /w/{code}.png` — всегда свежие обои (сегодняшняя точка закрашена) — её тянет ярлык iOS раз в день; счётчик скачиваний пишется в БД
- `GET /s/{code}` — страница установки: 3 шага (ярлык → автоматизация → проверка)
- `data/vita.db` — SQLite, создаётся сама (в git не входит)

## Мастер-ярлык «Vita» (создаётся один раз на iPhone)

1. Приложение «Команды» → «+» → назвать ярлык **Vita**
2. Добавить действия по порядку:
   - **Текст** — вставить любую ссылку вида `https://…/w/код.png` (заглушка)
   - **Получить содержимое URL** — вход: Текст
   - **Установить обои** — вход: Содержимое URL; выключить «Показывать предпросмотр»
3. Поделиться → включить **Import Question** для поля «Текст»
   (вопрос: «Вставь свою ссылку Vita») → **Скопировать ссылку iCloud**
4. Вставить ссылку в `SHORTCUT_ICLOUD_URL` в `app/main.py` — кнопка
   «Добавить ярлык» на странице установки оживёт сама.

## Воронка (цель продукта)

Reels → сайт → настройка 2–3 мин → оплата СБП → установка по инструкции ≤3 мин.
Максимальный минимализм, ноль регистраций.
