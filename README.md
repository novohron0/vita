# Vita — твоя жизнь в точках

Генератор обоев для айфона: месяц / год / жизнь в виде сетки точек.
Каждый день сервер закрашивает новую точку — обои обновляются сами через Ярлыки iOS.

## Запуск

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
.venv/bin/uvicorn app.main:app --port 8787
```

Открыть http://localhost:8787

## Как устроено

- `static/` — одна страница: настройка + живое превью на канвасе + скачивание PNG
- `app/render.py` — тот же рендер на Pillow для серверных картинок
- `POST /api/link` — сохраняет настройки, возвращает персональную ссылку
- `GET /w/{code}.png` — всегда свежие обои (сегодняшняя точка закрашена) — её тянет ярлык iOS раз в день
- `data/vita.db` — SQLite, создаётся сама (в git не входит)

## Воронка (цель продукта)

Reels → сайт → настройка 2–3 мин → оплата СБП → установка по инструкции ≤3 мин.
Максимальный минимализм, ноль регистраций.
